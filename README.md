### Installation (Linux)

- Just, Clone this repository (Linux) -
  ```
  git clone https://github.com/mohammmedVDX/VDXboomer.git
  ```
  
- Now Download Requirements (Linux) -
  ```
  cd VDXboomer
  pip install -r requirements.txt
  ```
  
- Now go to cloned directory and run `VDXboomer.py`(Linux) -
  ```
  cd VDXboomer
  python VDXboomer.py
  ```
